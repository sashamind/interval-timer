# Интервальный таймер

Интервальный таймер для тренировок со скакалкой (и любых HIIT-интервалов). Настраиваешь работу, отдых, количество раундов и подготовку — жмёшь «Начать».

## Возможности

- Настройка **работы**, **отдыха**, **количества раундов** и **подготовки** кнопками ±
- Пресеты: `30/30 ×12`, Табата `20/10 ×8`, `45/30 ×10`
- Крупный дисплей, видно с расстояния; фон меняет цвет по фазе (красный — работа, синий — отдых, жёлтый — подготовка, зелёный — финиш)
- Звуковые сигналы: отсчёт последних 3 секунд, разные тоны на старт работы/отдыха, аккорд на финише
- Вибрация на телефоне в конце
- Экран не гаснет во время тренировки (Wake Lock API)
- Пробел — пауза/продолжить
- Работает офлайн и ставится на телефон как приложение (PWA)

## Структура

```
index.html            # разметка
styles.css            # стили
app.js                # логика таймера
manifest.webmanifest  # манифест PWA
sw.js                 # service worker (офлайн-кэш)
icon.svg              # иконка приложения
```

## Локальный запуск

Service worker и Wake Lock требуют `https` или `localhost`. Просто открыть файл через `file://` — service worker не заработает (сам таймер будет). Проще всего поднять локальный сервер:

```bash
python3 -m http.server 8000
# открой http://localhost:8000
```

## Публикация на GitHub Pages

```bash
git init
git add .
git commit -m "Интервальный таймер"
git branch -M main
git remote add origin git@github.com:USERNAME/REPO.git
git push -u origin main
```

Затем в репозитории: **Settings → Pages → Source → Deploy from a branch → `main` / `root`**.
Через минуту таймер будет доступен по адресу `https://USERNAME.github.io/REPO/`.

> Если положишь таймер в **подпапку** существующего репозитория (например `USERNAME.github.io/timer/`), пути в проекте относительные — всё заработает без правок.

## Установка на телефон

Открой опубликованную ссылку в браузере телефона → меню → «Добавить на главный экран». Дальше запускается как отдельное приложение, полноэкранно и офлайн.

## Настройка под план тренировок

- Базовые интервалы: работа `30` / отдых `30` / раунды `12–15`
- Табата: работа `20` / отдых `10` / раунды `8`
- Пик-интервалы: работа `45` / отдых `30` / раунды `10`
